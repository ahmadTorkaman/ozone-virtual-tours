## Required Files

For the demo to work, add these files:

- `kitchen-overview.jpg` - Main kitchen panorama
- `kitchen-island.jpg` - Kitchen island view  
- `dining-area.jpg` - Dining area view
- `floorplan.png` - Floor plan image

## Where to Get Test Panoramas

1. **Render your own** from your 3D software (V-Ray, Corona, etc.)
2. **Download free samples** from:
   - https://polyhaven.com/hdris (Convert HDRI to equirectangular JPEG)
   - https://www.flickr.com/search/?q=equirectangular (Search Creative Commons)
   - https://360cities.net (Some free downloads available)

## Image Requirements

- **Format**: JPEG or PNG
- **Aspect Ratio**: 2:1 (equirectangular projection)
- **Recommended Size**: 4096 x 2048 pixels
- **Max File Size**: 50MB

## Floor Plan

The floor plan should be a simple top-down view showing room layouts.
Any image format works (PNG recommended for transparency).
